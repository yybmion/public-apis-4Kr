# 📋 Public APIs 기능 설명서

이 문서는 각 API가 **어떤 데이터를 제공하고**, **어떤 기능을 사용할 수 있는지** 한눈에 볼 수 있도록 정리한 설명서입니다.

> 코드 예제가 필요하신 경우 [API 사용 예제 가이드](./example_usage.md)를 참고하세요.

## 🏛 정부 & 공공기관

### 공공데이터포털
**제공 기관:** 행정안전부
**웹사이트:** https://www.data.go.kr/

**제공 데이터:**
- 전국의 공공 데이터 10만개 이상 통합 제공
- 실시간 교통 정보, 날씨, 대기질, 재난 정보
- 의료기관 위치 및 정보
- 문화시설, 관광지 정보
- 부동산 실거래가
- 공공시설 위치 (도서관, 체육관, 주차장 등)

**사용 가능한 기능:**
- 지역별 미세먼지 및 대기오염 수치 조회
- 전국 병원, 약국 위치 및 진료 과목 검색
- 아파트 매매/전월세 실거래가 조회
- 문화재, 박물관, 미술관 정보 검색
- 공공 와이파이 위치 조회
- 재난 안전 정보 실시간 확인

### 기상청 단기예보
**제공 기관:** 기상청
**웹사이트:** https://www.data.go.kr/data/15084084/openapi.do

**제공 데이터:**

- 초단기 예보 (6시간 이내)

**사용 가능한 기능:**

- 시간대별 날씨 예보 조회
- 지역별 강수 확률 확인

### 한국관광공사 TourAPI
**제공 기관:** 한국관광공사
**웹사이트:** https://api.visitkorea.or.kr/

**사용 가능한 기능:**

- 현재 진행 중인 축제 확인
- 테마별 여행 코스 조회
- 관광지 상세 정보 (운영시간, 입장료, 주차 정보)

- 카테고리별 주변 검색 (음식점, 카페, 주차장 등)
- 두 지점 간 거리 계산
- 자동차, 도보 경로 탐색
- 지도 위에 마커 표시

## 📊 추가 정보

### 데이터 갱신 주기

| 카테고리 | 데이터 | 갱신 주기 |
|---------|--------|---------|
| 교통 | 버스/지하철 도착 정보 | 실시간 (1분) |
| 날씨 | 초단기 예보 | 10분 |
| 환경 | 미세먼지 | 1시간 |
| 금융 | 주식 시세 | 실시간 |
| 부동산 | 실거래가 | 월 1회 |
| 통계 | 정부 통계 | 분기별 |

### 사용 시 유의사항

1. **개인정보 보호**
   - 금융, 의료 데이터는 본인 정보만 조회 가능
   - 사용자 동의 필수

2. **상업적 이용**
   - 대부분 API는 상업적 이용 가능
   - 일부 API는 비영리 목적만 허용 (서울시 TOPIS 등)

3. **호출 제한**
   - 대부분 일 1,000~10,000건 제한
   - 초과 시 일시적 차단

4. **데이터 저장**
   - 캐싱 목적의 단기 저장은 허용
   - 대량 수집 후 재판매는 금지
