## 2. 초보자가 알아야 할 주식 기본 용어

### 2.1 기본 개념

#### 주식이란?
> 회사의 소유권을 나눈 조각. 주식을 사면 그 회사의 주인이 됩니다.

**예시**: 삼성전자 주식 1주 = 삼성전자의 아주 작은 주인

#### 주가 (주식 가격)
> 주식 1개의 가격

**예시**: 삼성전자 75,000원 → 주식 1개를 사려면 75,000원 필요

#### 시가총액
> 회사 전체의 가치 (주가 × 주식 수)

**비유**:
- 시가총액 큼 = 대형 마트 (안전함)
- 시가총액 작음 = 동네 슈퍼 (위험하지만 성장 가능)

### 2.2 재무 지표 (쉽게 설명)

#### PER (주가수익비율)
> 주가가 1년 순이익의 몇 배인가?

**쉽게**: PER 15배 = "이 회사가 1년에 1억 버는데, 주가는 15억"

**판단**:
- PER 낮음 (< 10) = 저평가 (싸다)
- PER 높음 (> 20) = 고평가 (비싸다, 하지만 성장 기대)

#### PBR (주가순자산비율)
> 주가가 회사 자산의 몇 배인가?

**쉽게**: PBR 1배 = "회사가 가진 땅, 건물, 돈이 1억인데, 주가도 1억"

**판단**:
- PBR < 1 = 저평가 (자산보다 싸게 거래)
- PBR > 2 = 고평가

#### ROE (자기자본이익률)
> 회사가 자기 돈으로 얼마나 잘 버는가? (%)

**쉽게**: ROE 15% = "회사가 100만원 가지고 1년에 15만원 벌었다"

**판단**:
- ROE > 15% = 우수
- ROE > 10% = 양호
- ROE < 5% = 부진

#### 배당수익률
> 주식을 1년 보유하면 받는 현금 (%)

**쉽게**: 배당수익률 3% = "1억 투자하면 매년 300만원 받음"

**판단**:
- > 3% = 높음 (좋음)
- 1~2% = 보통
- 0% = 배당 없음

### 2.3 차트 용어

#### 이동평균선 (MA)
> 최근 N일 동안의 평균 주가

**예시**:
- 20일 이동평균선 = 최근 20일 평균 가격
- 주가가 이평선 위 = 상승 추세 ↗️
- 주가가 이평선 아래 = 하락 추세 ↘️

#### 골든크로스 / 데드크로스
**골든크로스** ✨
```
단기 이평선(5일)이 장기 이평선(20일)을 뚫고 올라감
→ 매수 신호!
```

**데드크로스** ☠️
```
단기 이평선이 장기 이평선 아래로 떨어짐
→ 매도 신호!
```

#### RSI (과매수/과매도)
> 주식이 너무 많이 올랐는지, 떨어졌는지 (0-100)

**판단**:
- RSI > 70 = 과매수 (너무 올라서 곧 떨어질 수 있음)
- RSI < 30 = 과매도 (너무 떨어져서 반등 가능)
- RSI 30-70 = 정상 범위

#### 거래량
> 하루에 거래된 주식 수

**중요한 이유**:
- 거래량 급증 + 주가 상승 = 강한 매수세 (좋은 신호)
- 거래량 급증 + 주가 하락 = 강한 매도세 (나쁜 신호)

## 3. 시스템 사용 방법 (5분 만에 시작)

### Step 2: 나의 투자 성향 파악하기

화면에 나오는 **5가지 질문**에 답하세요.

#### 질문 1: 투자 가능 금액은?
- [ ] 500만원 미만
- [ ] 500~1,000만원
- [ ] 1,000~3,000만원
- [ ] 3,000만원 이상

**팁**: 없어도 생활에 지장 없는 돈으로만 투자하세요!

#### 질문 2: 투자 목표 기간은?
- [ ] 1개월 이내 (단기) → 추천하지 않음 (도박에 가까움)
- [ ] 3~6개월 (중기)
- [ ] 1년 이상 (장기) → 가장 안전

**팁**: 주식은 장기 투자가 유리합니다!

#### 질문 3: 투자 원금의 -10% 손실 발생 시?
- [ ] 매우 불안하다 → **리스크 LOW** (안전한 종목 추천)
- [ ] 견딜 수 있다 → **리스크 MEDIUM**
- [ ] 기회로 본다 → **리스크 HIGH** (초보자 비추천)

#### 질문 4: 선호하는 투자 스타일?
- [ ] 안정적인 배당주 (매년 현금 받기)
- [ ] 성장하는 기술주 (큰 수익 기대)
- [ ] 저평가된 가치주 (싸게 사서 비싸게 팔기)

#### 질문 5: 관심 섹터는? (복수 선택)
- [ ] IT/반도체 (삼성전자, SK하이닉스)
- [ ] 금융 (은행, 증권사)
- [ ] 헬스케어 (제약, 바이오)
- [ ] 필수소비재 (식품, 생활용품)
- [ ] 에너지
- [ ] **모르겠음** ← 이거 선택하면 시스템이 자동 추천!

## 4. 추천 종목은 어떻게 선택되나요?

### 추천 기준 (초보자 안전 우선)

#### 1단계: 기본 필터링
```
✅ 시가총액 1조원 이상
   → 대형주만 선택 (안정성)

✅ 일평균 거래대금 100억 이상
   → 언제든 사고 팔 수 있음 (유동성)

✅ 부채비율 200% 이하
   → 빚이 많지 않음 (재무 안정성)

✅ ROE 10% 이상
   → 돈을 잘 버는 회사

✅ 외국인 보유 20% 이상
   → 외국 기관도 신뢰하는 회사

✅ 최근 3년 연속 흑자
   → 적자 나는 회사 제외
```

#### 2단계: 초보자 적합도 점수 계산 (0-100점)

| 항목 | 배점 | 예시 |
|------|------|------|
| **시가총액** | 30점 | 10조 이상 → 30점 |
| **변동성** | 20점 | 낮을수록 높은 점수 |
| **ROE** | 20점 | 15% 이상 → 20점 |
| **배당수익률** | 15점 | 3% 이상 → 15점 |
| **외국인 보유** | 15점 | 30% 이상 → 15점 |

**점수 해석**:
- 80점 이상 ★★★★★ → 매우 추천 (초보자에게 최적)
- 70~79점 ★★★★☆ → 추천
- 60~69점 ★★★☆☆ → 보통
- 60점 미만 → 초보자에게 비추천

#### 3단계: 미국 시장 신호 반영

**중요!** 미국 증시와 한국 증시는 **상관계수 0.85** (매우 강한 연관)

```
S&P 500 > 20일 이동평균선
→ 🟢 BULLISH (한국 주식 매수 유리)

S&P 500 < 20일 이동평균선
→ 🔴 BEARISH (한국 주식 조심, 현금 보유)
```

**백테스팅 결과** (2018-2023):
- S&P 500 신호 활용 전략: **연평균 12.3% 수익**
- 코스피 단순 보유: 연평균 5.8% 수익
- **→ 2배 이상 차이!**

## 5. 미국 증시가 왜 중요한가요?

### 한마디 요약

> **"미국이 콜록거리면, 한국은 독감에 걸린다"**

### 과학적 근거

#### 상관계수 0.85의 의미
```
상관계수 0.85 = 미국 증시가 1% 움직이면,
                한국 증시도 약 0.85% 같은 방향으로 움직임
```

**역사적 데이터** (1980-2023):
- S&P 500 ↗️ 1% → 코스피 평균 ↗️ 0.8%
- S&P 500 ↘️ 1% → 코스피 평균 ↘️ 0.9%

### 왜 이런 일이?

#### 1. **시차 효과**
```
미국 증시 (전날 밤 23:30 마감)
    ↓
한국 증시 (다음날 09:00 개장)
    ↓
미국이 크게 떨어지면 → 한국도 하락 출발
```

#### 2. **무역 의존도**
- 한국 수출 1위: 중국 (25%)
- 한국 수출 2위: **미국 (15%)**
- 미국 경기 나쁨 → 한국 기업 매출 감소

#### 3. **외국인 투자**
```
미국 경기 좋음
→ 외국인 돈이 한국으로 유입
→ 한국 주식 상승

미국 경기 나쁨
→ 외국인 돈 빠져나감
→ 한국 주식 하락
```

### 실전 활용법

#### S&P 500 확인 방법
```
시스템 메인 화면 → [시장 현황] 탭
┌─────────────────────────────────────┐
│ 🇺🇸 미국 시장 현황                   │
├─────────────────────────────────────┤
│ S&P 500: 4,550 (-1.3%) ↘️            │
│ 20일 이평선: 4,600                   │
│                                     │
│ 현재 신호: 🔴 BEARISH                │
│                                     │
│ ⚠️ 주의: S&P 500이 이평선 아래입니다. │
│   한국 증시도 조정 가능성 있습니다.    │
│   신규 매수는 신중하게!               │
└─────────────────────────────────────┘
```

#### 신호별 대응 전략

**🟢 BULLISH (S&P 500 > 이평선)**
```
✅ 추천 종목 적극 매수
✅ 포트폴리오 비중 증가 (현금 → 주식)
✅ 공격적 투자 가능
```

**🔴 BEARISH (S&P 500 < 이평선)**
```
⚠️ 신규 매수 보류
⚠️ 보유 종목 일부 매도 고려
⚠️ 현금 비중 증가 (50% 이상)
```

**⚪ NEUTRAL (이평선 근처)**
```
△ 관망 (지켜보기)
△ 소량만 매수
```

## 7. 차트 보는 법 (기초)

### 캔들스틱 차트 해석

#### 캔들 하나의 의미

```
    │  ← 고가 (최고로 올라간 가격)
    ┃
  ┏━┓
  ┃ ┃ ← 시가 (시작 가격)
  ┃ ┃    종가 (마지막 가격)
  ┗━┛
    ┃
    │  ← 저가 (최저로 떨어진 가격)
```

#### 빨간 캔들 (상승)
```
    │
  ┏━┓ 빨간색
  ┃ ┃ 종가 > 시가
  ┗━┛ → 상승한 날
    │
```

#### 파란 캔들 (하락)
```
    │
  ┏━┓ 파란색
  ┃ ┃ 종가 < 시가
  ┗━┛ → 하락한 날
    │
```

### 차트 패턴 (자주 보이는 것만)

#### 1. 상승 추세
```
        ／
      ／
    ／
  ／
```
**특징**: 빨간 캔들 많음, 주가가 이평선 위
**판단**: ✅ 매수 고려

#### 2. 하락 추세
```
＼
  ＼
    ＼
      ＼
```
**특징**: 파란 캔들 많음, 주가가 이평선 아래
**판단**: ⚠️ 관망 또는 매도

#### 3. 횡보 (박스권)
```
━━━━━  ← 저항선
～～～～
━━━━━  ← 지지선
```
**특징**: 일정 가격대에서 왔다 갔다
**판단**: △ 돌파 기다리기

### 이동평균선으로 판단하기

#### 배열 (정배열 vs 역배열)

**정배열 (상승 추세)** ✅
```
주가 ─────────────
5일선 ─────────
20일선 ───────
60일선 ─────

→ 매수 신호
```

**역배열 (하락 추세)** ⚠️
```
60일선 ─────
20일선 ───────
5일선 ─────────
주가 ─────────────

→ 매도 신호
```

### 거래량의 비밀

```
┃           ┃
┃     ┃     ┃
┃  ┃  ┃  ┃  ┃
━━━━━━━━━━━━━━  ← 시간축

주가 ↗️ + 거래량 ↗️ = 🟢 매수 신호 (강한 상승)
주가 ↗️ + 거래량 ↘️ = ⚠️ 약한 상승 (곧 하락?)
주가 ↘️ + 거래량 ↗️ = 🔴 매도 신호 (강한 하락)
```

### 초보자를 위한 차트 체크리스트

매수 전 확인:
- [ ] 주가가 20일 이평선 위에 있는가?

**2개 이하 YES → 관망**

## 8. 리스크 관리 (손실 줄이는 법)

### 손절매 (Stop Loss) - 가장 중요!

#### 손절매란?
> 손실을 일정 수준에서 끊는 것

**비유**: "배가 침몰하기 전에 탈출!"

#### 손절매 기준

**기본 규칙**: 매수가 대비 **-5% 하락 시 무조건 매도**

**예시**:
```
삼성전자를 75,000원에 매수
손절가 = 75,000 × 0.95 = 71,250원

현재가가 71,250원 이하로 떨어지면
→ 즉시 매도! (감정 개입 금지)
```

**왜 -5%?**
- 😢 -10% 손실 회복하려면 +11% 필요
- 😭 -20% 손실 회복하려면 +25% 필요
- 💀 -50% 손실 회복하려면 +100% 필요

**→ 작은 손실에 끊어야 큰 손실 방지!**

#### 시스템에서 자동 알림 설정

```
[알림 설정] → [손절가 알림]
손절가: 71,250원 입력
☑️ 알림 ON

→ 손절가 도달 시 카카오톡 알림!
```

### 포지션 크기 조절

#### 한 종목에 얼마나 투자?

**기본 규칙**:
```
한 종목에 최대 20%까지만 투자
```

**예시 (1,000만원)**:
```
✅ 좋은 예:
  - 삼성전자: 200만원 (20%)
  - KB금융: 150만원 (15%)
  - CJ제일제당: 150만원 (15%)
  - 기타 종목: 각 100만원씩
  - 현금: 100만원 (10%)

❌ 나쁜 예:
  - 삼성전자: 500만원 (50%) ← 위험!
  - 기타: 500만원
```

**왜?** 한 종목이 폭락해도 전체 손실은 -10% 이내

#### 리스크 수준별 투자 비중

| 리스크 | 최대 투자 비중 | 예시 |
|--------|---------------|------|
| **LOW** | 20% | 삼성전자, CJ제일제당 |
| **MEDIUM** | 15% | 네이버, 셀트리온 |
| **HIGH** | 5% | 중소형주 (초보자 비추천) |

### 분산 투자

#### 왜 분산?

**집중 투자**:
```
삼성전자 100% 투자
→ 삼성전자 -20% = 내 자산 -20% 😭
```

**분산 투자**:
```
삼성전자 20%, KB금융 20%, CJ제일제당 20%, ...
→ 삼성전자 -20% = 내 자산 -4% 😊
```

#### 분산 전략

**최소 5개 종목 보유**

```
✅ 좋은 분산:
  - IT/반도체 30% (삼성전자, NAVER)
  - 금융 20% (KB금융)
  - 필수소비재 20% (CJ제일제당)
  - 헬스케어 20% (셀트리온)
  - 현금 10%

❌ 나쁜 분산:
  - IT/반도체 70% (삼성전자, SK하이닉스, NAVER)
    → 같은 섹터 = 같이 떨어짐
```

### 절대 하지 말아야 할 것

#### 1. 물타기 ❌
```
잘못된 예:
삼성전자 75,000원에 매수
↓ -10% 하락
67,500원에 추가 매수 (평단가 낮추기)
↓ -20% 더 하락
→ 큰 손실 😭

올바른 예:
삼성전자 75,000원에 매수

```

#### 2. 빚내서 투자 (레버리지) ❌
```
1,000만원 빌려서 투자
→ 이자 부담 + 원금 상환 압박
→ 급락 시 강제 청산
```

**초보자는 절대 레버리지 금지!**

#### 3. 단타 (하루에 여러 번 매매) ❌
```
단타 = 도박에 가까움
수수료 + 세금 = 연간 -10% 이상
```

**초보자는 최소 3개월 이상 보유 추천**

#### 4. 소문 믿고 투자 ❌
```
"카페에서 000 대박 난다더라"
"유튜브에서 000 강력 추천"
```

### Q4. 수익률 목표는?

**A**:

**현실적인 목표**:
- 1개월: +3~5%
- 3개월: +10~15%
- 1년: +20~30%

**비교**:
- 은행 예금 금리: 연 3~4%
- 코스피 평균: 연 5~8%
- 이 시스템 백테스팅: **연 12.3%**

**주의**: 연 50% 이상 목표 = 너무 욕심
**→ 리스크 증가, 손실 확률 ↑**

### Q7. 손실이 나면 어떡하죠?

**A**:

**손절매 규칙 준수!**

```
-5% 손실 → 즉시 매도
감정 개입 금지!
```

**손실은 자연스러운 것**:
- 초보자 평균 승률: 50% (10번 중 5번 손실)
- 중요한 건 **손실을 작게, 수익을 크게**

**예시**:
```
10번 투자
- 6번 성공: +10% × 6 = +60%
- 4번 실패: -5% × 4 = -20%
총: +40% 수익
```

**절대 평균 단가 낮추기(물타기) 하지 마세요!**

### Q8. 시스템 추천이 틀릴 수도 있나요?

**A**: 네, **당연히 가능합니다**.

**이유**:
- 예측은 100% 아님 (과거 데이터 기반)
- 갑작스러운 악재 (전쟁, 천재지변)
- 시장은 비합리적으로 움직일 수 있음

**하지만**:
- 백테스팅 승률: 60% (10번 중 6번 성공)
- 손절매로 손실 최소화
- 장기적으로 수익

**→ 단기 손실에 흔들리지 마세요!**
