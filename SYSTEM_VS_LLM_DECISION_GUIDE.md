# 수익 창출 관점: 시스템 vs LLM 판단 구간 가이드

## 📊 개요

본 문서는 Stock Intelligence System에서 **수익을 창출할 수 있는 조건**을 기준으로, 시스템(규칙 기반 알고리즘)이 판단해야 하는 구간과 LLM(대규모 언어 모델)이 판단하면 좋은 구간을 구분합니다.

---

## ✅ 시스템이 판단해야 하는 구간 (현재 코드로 충분)

### 1. 기술적 지표 계산 및 시그널 생성 ⚡

**현재 위치**: `TechnicalAnalyzer`, `SignalDetector`

**판단 이유**:
- 수학적 계산은 정확성이 생명 → LLM 불필요
- 실시간 대응 필요 (골든크로스 발생 즉시 알림)
- 백테스팅으로 검증 가능한 명확한 룰

**코드 예시**:
```python
# RSI 70 이상 → 과매수 (시스템 판단 ✅)
if rsi > 70:
    signal = "OVERBOUGHT"
    
# MA(5) > MA(20) → 골든크로스 (시스템 판단 ✅)
if ma_5 > ma_20:
    signal = "GOLDEN_CROSS"

# MACD 크로스오버 감지 (시스템 판단 ✅)
if macd > macd_signal and prev_macd < prev_macd_signal:
    signal = "BUY"
```

**수익 기여도**: ★★★★★ (핵심)

**장점**:
- 밀리초 단위 빠른 계산
- 일관성 보장 (같은 입력 = 같은 출력)
- 백테스팅으로 성과 검증 가능
- 비용 제로 (API 호출 불필요)

**단점**:
- 시장 맥락 이해 불가
- 뉴스/이벤트 반영 안됨

---

### 2. 미국 시장 상관관계 판단 🇺🇸

**현재 위치**: `SignalDetector.get_us_market_signal()`

**판단 이유**:
- 0.85 상관계수는 통계적 사실
- S&P 500 > MA(20) = BULLISH는 명확한 룰
- 속도가 중요 (시장 개장 전 판단 필요)

**코드 예시**:
```python
# S&P 500이 MA(20) 위 → 한국 주식 매수 (시스템 판단 ✅)
if sp500_close > sp500_ma20:
    korean_market_bias = "BULLISH"
    recommended_action = "INCREASE_EQUITY"
    confidence = min(100, 50 + abs(diff_pct) * 10)
    
# 이격도 계산
ma_diff_pct = ((sp500_close - sp500_ma20) / sp500_ma20) * 100

# 한국 주식 점수 조정
if us_signal == 'BULLISH':
    korean_stock_score += us_confidence * 0.3  # 상관계수 0.85 반영
```

**수익 기여도**: ★★★★☆ (매우 중요)

**실전 효과**:
- 미국 시장 선행성 활용 (한국 개장 전 방향성 파악)
- 리스크 회피 (미국 약세 시 포지션 축소)
- 글로벌 자금 흐름 추적

---

### 3. 재무제표 점수화 💰

**현재 위치**: `BeginnerRecommender.calculate_score()`

**판단 이유**:
- ROE, PER, 배당수익률은 객관적 숫자
- 가중치 기반 점수는 백테스팅으로 최적화 가능
- 일관성 필요 (같은 재무제표 = 같은 점수)

**코드 예시**:
```python
# ROE 20% 이상 → 20점 (시스템 판단 ✅)
if roe >= 20:
    score += 20
elif roe >= 15:
    score += 15
elif roe >= 10:
    score += 10
    
# 시가총액 10조 이상 → 30점 (시스템 판단 ✅)
if market_cap > 10_000_000_000_000:
    score += 30
elif market_cap > 5_000_000_000_000:
    score += 20
    
# 배당수익률 4% 이상 → 15점 (시스템 판단 ✅)
if dividend_yield >= 4.0:
    score += 15
elif dividend_yield >= 2.5:
    score += 10
```

**수익 기여도**: ★★★★☆ (장기 수익)

**벤치마크 가능**:
- 과거 5년 ROE 20% 이상 종목의 평균 수익률 추적
- 배당 재투자 효과 계산
- 가치 투자 전략 백테스팅

---

### 4. 백테스팅 성과 측정 📈

**현재 위치**: `BacktestEngine`

**판단 이유**:
- Sharpe Ratio, MDD, CAGR는 수학 공식
- 과거 데이터 기반 검증은 LLM 불필요
- 객관적 전략 평가 필요

**코드 예시**:
```python
# Sharpe > 1.0 and MDD < -25% → 합격 (시스템 판단 ✅)
if sharpe_ratio > 1.0 and mdd > -25:
    strategy_grade = "GOOD"
    deploy_to_production = True
    
# CAGR vs 벤치마크 비교
if cagr > kospi_cagr + 5:
    alpha = cagr - kospi_cagr
    outperformance = True
```

**수익 기여도**: ★★★★★ (전략 검증)

**핵심 지표**:
- **Sharpe Ratio > 1.0**: 위험 대비 수익 우수
- **MDD < 25%**: 심리적 손실 감내 가능
- **Win Rate > 50%**: 전략 신뢰성
- **CAGR > 10%**: 장기 복리 효과

---

### 5. 리스크 레벨 분류 ⚖️

**현재 위치**: `BeginnerRecommender.classify_risk()`

**판단 이유**:
- 변동성, 베타, 시가총액은 계산 가능
- 명확한 기준선 설정 가능
- 법적 책임 이슈 (객관적 기준 필요)

**코드 예시**:
```python
# 변동성 기반 위험도 (시스템 판단 ✅)
if volatility < 1.0 and market_cap > 10T:
    risk_level = "LOW"
elif volatility < 2.0 and market_cap > 1T:
    risk_level = "MEDIUM"
else:
    risk_level = "HIGH"
```

**수익 기여도**: ★★★☆☆ (리스크 관리)

## 🤖 LLM이 판단하면 좋은 구간

### 1. 뉴스 기반 리스크 평가 📰 (최우선 추천)

**현재 위치**: `SentimentAnalyzer` → **LLM으로 강화 필요**

**판단 이유**:
- "삼성전자 신규 파운드리 수주" vs "삼성전자 반도체 경쟁 심화" 
  → 같은 긍정/부정도 뉴앙스 차이 큼
- 복합적 뉴스 해석 (예: 실적 호조 but 경쟁사 더 좋음)
- 숨겨진 리스크 감지 (예: 경영진 교체, 소송, 정책 변화)

**현실 예시**:
```
뉴스 1: "삼성전자, 3분기 영업이익 15조원 달성"
뉴스 2: "SK하이닉스, HBM3 점유율 90% 돌파"
뉴스 3: "미국, 중국 반도체 수출 규제 강화"

→ 시스템 판단: 긍정 2개, 중립 1개 = 종합 긍정 ❌
   (단순 키워드 카운팅)

→ LLM 판단: ✅
   "삼성전자의 실적은 양호하나, SK하이닉스가 HBM 시장을 
    장악하며 차세대 메모리 경쟁에서 뒤처지는 모습입니다.
    
    미국의 중국 수출 규제는 삼성전자에도 간접 타격 예상.
    중국 매출 비중 20%를 고려하면 향후 실적 불확실성 증가.
    
    종합 판단: 단기 실적은 호조이나, 구조적 경쟁력 약화 우려.
    신규 매수보다는 기존 보유 물량 일부 익절 권장."
```

**수익 기여도**: ★★★★★ (리스크 회피)

**LLM 활용 포인트**:
- **맥락 이해**: 단순 긍정/부정이 아닌 상대적 평가
- **연관성 분석**: 경쟁사 뉴스와 비교
- **시간 축**: 단기 호재 vs 장기 악재 구분
- **암묵적 정보**: "연기", "검토 중" 같은 부정 암시 포착

---

### 2. 복합 신호의 우선순위 결정 🎯

**현재 위치**: `SignalDetector.generate_combined_signal()` → **LLM으로 강화**

**판단 이유**:
- 기술적 지표 강세 but 뉴스 부정적 → 어느 쪽 우선?
- 단기 vs 장기 신호 충돌 시 판단
- 섹터 전망, 거시 경제 등 비정형 요소 통합

**현실 예시**:
```
상황:
- 기술적 신호: 골든크로스 발생 (매수)
- 미국 시장: S&P 500 MA 위 (긍정)
- 뉴스 감성: 부정적 70%
- 재무: ROE 5% (약함)
- 섹터: IT 반도체 경기 하락 중

→ 시스템: 2개 긍정, 2개 부정 = 중립 (HOLD) ❌
   (단순 가중 평균)

→ LLM: ✅
   "골든크로스는 발생했으나, 이는 후행 지표입니다.
    
    현재 더 중요한 요소:
    1. 뉴스 70% 부정 → 펀더멘털 악화 신호
    2. ROE 5% → 수익성 저하
    3. 반도체 경기 사이클 하락 중
    
    골든크로스는 '데드캣 바운스(일시 반등)' 가능성.
    과거 유사 사례(2022년 10월)에서도 골든크로스 후
    2주 뒤 -15% 하락 경험.
    
    권장: 골든크로스를 매수 신호가 아닌 청산 기회로 활용.
           반등 시 기존 보유 물량 정리 후 관망."
```

**수익 기여도**: ★★★★★ (타이밍)

**LLM 활용 포인트**:
- **선행/후행 구분**: 기술적 지표의 시간 지연 고려
- **과거 유사 사례**: 패턴 매칭
- **우선순위 동적 조정**: 시장 국면별 중요 요소 변경

---

### 3. 비정형 리스크 요인 발견 ⚠️

**현재 위치**: 없음 → **LLM 신규 추가 필요**

**판단 이유**:
- 경영진 스캔들, 회계 이슈, 공급망 문제 등
- 뉴스에 암시적으로 나타나는 위험 신호
- 업종 특수성 (예: 바이오는 임상 실패 리스크)

**현실 예시**:

**케이스 1: 바이오 종목**
```
뉴스: "A제약, 3상 임상 중간 데이터 공개 연기"

→ 시스템: 중립 뉴스 (긍정/부정 키워드 없음) ❌

→ LLM: ✅
   "바이오 섹터에서 '연기'는 70% 확률로 부정적 결과를 암시합니다.
    
    과거 사례:
    - B제약(2021): 연기 3개월 후 임상 실패 발표
    - C제약(2020): 연기 후 FDA 승인 거절
    
    현재 A제약 시가총액의 80%가 이 파이프라인에 베팅된 상황.
    실패 시 주가 -50% 예상.
    
    권장: 임상 결과 공식 발표까지 신규 매수 보류.
           기존 보유자는 손절 검토 (리스크 대비 기대수익 불리)."
```

**케이스 2: 경영진 이슈**
```
뉴스: "D그룹 회장, 검찰 소환 조사"

→ 시스템: 부정 뉴스로 분류하나 영향도 미측정 ❌

→ LLM: ✅
   "D그룹은 오너 경영 체제로, 회장의 의사결정이 절대적.
    
    과거 유사 사례:
    - E그룹(2019): 회장 구속 후 6개월간 -35% 하락
    - F그룹(2017): 재판 장기화로 투자 지연, -20% 하락
    
    D그룹의 경우 주력 사업이 반도체로, 대규모 투자 결정이
    향후 3년 실적을 좌우. 회장 부재 시 투자 지연 불가피.
    
    권장: 법적 이슈 해소까지 보수적 접근.
           같은 섹터 대체 종목(SK하이닉스 등) 고려."
```

**수익 기여도**: ★★★★★ (손실 방지)

**LLM 활용 포인트**:
- **도메인 지식**: 업종별 리스크 패턴 학습
- **과거 사례 매칭**: 유사 상황의 결과 예측
- **간접 영향 추론**: A사 이슈 → B사 수혜 연결

### 4. 초보자 맞춤 설명 생성 💡

**현재 위치**: `BeginnerRecommender.generate_reasons()` → **LLM 강화**

**판단 이유**:
- "ROE 20%"를 초보자가 이해 못함
- 같은 데이터라도 투자자 수준별로 다른 설명 필요
- 행동 유도 (예: "지금 사세요" vs "더 지켜보세요")

**현실 예시**:

**시스템 출력** (현재):
```
추천 종목: 삼성전자 (005930)
- 점수: 85/100
- ROE: 20%
- PER: 12
- 배당수익률: 3.5%
- 위험도: LOW
- 추천 이유: 시가총액 대형, 재무 우수, 배당 안정
```

**LLM 강화 출력**:
```
🎯 삼성전자 - 초보자 강력 추천 (85점)

📊 왜 좋은가요?
"삼성전자는 투자한 돈으로 매년 20%의 수익을 내는 회사입니다(ROE).
 은행 예금 금리가 3%인 것과 비교하면 6배 이상 높은 수익력이죠.
 
 현재 주가는 이 좋은 회사를 저평가하고 있어요(PER 12).
 같은 실적을 내는 미국 회사들은 PER 20 수준인데 말이죠.
 
 게다가 1년에 3.5%의 배당도 줍니다.
 100만원 투자하면 매년 3만 5천원씩 받으시는 거예요."

💰 얼마나 벌 수 있나요?
예상 수익률: 1년 +15~25% (배당 포함)
- 주가 상승: +10~20% (현재 저평가 해소)
- 배당 수익: +3.5%
- 미국 시장 호조 시 추가 상승 가능

📅 언제 사야 하나요?
지금 사도 좋지만, 더 좋은 타이밍:
1. 다음 주 화요일 (미국 빅테크 실적 발표 후)
2. 월말 배당 기준일 2주 전

⚠️ 주의사항
- 단기 급등은 기대하기 어려워요 (안정적 대형주)
- 최소 1년 이상 보유 권장
- 반도체 경기가 나빠지면 일시적으로 -10% 하락 가능
  → 이 경우 추가 매수 기회로 활용

🔰 초보자 팁
- 한 번에 다 사지 말고 3번에 나눠 매수
- 목표가: 85,000원 (현재 대비 +20%)
- 손절가: 60,000원 (현재 대비 -15%)
```

**수익 기여도**: ★★★☆☆ (교육 효과, 간접 기여)

**LLM 활용 포인트**:
- **쉬운 비유**: "ROE 20% = 은행 금리 6배"
- **구체적 금액**: "100만원 → 3만 5천원"
- **행동 지침**: "언제, 얼마나, 어떻게"
- **심리 관리**: 하락 시 대응 방법

### 5. 시나리오 플래닝 🔮

**현재 위치**: 없음 → **LLM 신규 기능**

**판단 이유**:
- "만약 금리가 0.5% 오르면?" 같은 가정 분석
- 최악/최선 시나리오별 대응 전략
- 포트폴리오 리밸런싱 권고

**현실 예시**:

**현재 포트폴리오**: 삼성전자 50%, SK하이닉스 30%, 현금 20%

**LLM 시나리오 분석**:
```
📊 포트폴리오 위험 진단

현재 상황:
- 반도체 집중도: 80% (위험!)
- 섹터 다각화: 부족
- 현금 비중: 20% (적정)
```

## 💡 핵심 원칙

### 시스템이 잘하는 것 (Fast & Accurate)
1. **계산**: 수학, 통계, 지표
2. **속도**: 실시간 신호 감지
3. **일관성**: 같은 입력 → 같은 출력
4. **객관성**: 감정 배제, 백테스팅 가능

### LLM이 잘하는 것 (Smart & Contextual)
1. **맥락**: 복합 정보 통합 이해
2. **뉴앙스**: 단순 긍정/부정 넘어선 해석
3. **과거 학습**: 유사 사례 매칭
4. **설명**: 전문 용어 → 쉬운 설명

### 수익 극대화 공식
```
수익 = 타이밍(시스템) × 리스크 회피(LLM) × 실행력(교육)

타이밍: 기술적 지표로 진입/청산 시점 포착
리스크 회피: LLM으로 숨겨진 위험 감지
실행력: 초보자도 이해할 수 있는 명확한 가이드
```
