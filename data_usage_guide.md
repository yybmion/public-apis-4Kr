## 🏛 정부 & 공공기관 데이터

### 공공데이터포털
**제공 데이터:**
- 10만개 이상의 다양한 데이터셋 (정부/공공기관 통합)
- 실시간 교통정보, 날씨, 환경, 의료, 복지, 문화 등 전 분야

**데이터 범위:**
- 국토교통부, 환경부, 행안부, 문체부 등 중앙부처 데이터
- 서울, 경기, 부산 등 지자체 데이터
- 역사적 데이터 및 실시간 데이터

**사용 제한:**
- **호출 제한**: API별로 상이 (일반적으로 일 1,000~10,000건)
- **상업적 이용**: ✅ 가능 (단, 데이터별 이용약관 확인 필요)
- **재배포**: ⚠️ 제한적 허용 (출처 표기 필수)
- **비용**: 무료

**주의사항:**
- 일부 데이터는 승인 후 1~2시간 대기 필요
- 인코딩/디코딩된 두 가지 키 제공
- 데이터 갱신 주기가 API마다 다름 (실시간 ~ 월 단위)

---

### 통계청 KOSIS
**제공 데이터:**
- 134,586종의 국가통계 데이터
- 인구, 경제, 사회, 문화 등 전 분야 통계

**데이터 범위:**
- 1960년대부터 현재까지의 시계열 데이터
- 전국/시도/시군구 단위 지역 통계
- 연령별, 성별, 직업별 등 다차원 분류

**사용 제한:**
- **호출 제한**: 일 10,000건
- **상업적 이용**: ✅ 가능
- **학술/연구**: ✅ 적극 권장
- **비용**: 무료

---

### 기상청 API (단기예보/중기예보)
**제공 데이터:**
- **초단기실황**: 현재 기온, 강수량, 습도, 풍속 등
- **초단기예보**: 6시간 이내 예보 (10분 단위 갱신)
- **단기예보**: 3일 이내 예보 (3시간 간격)
- **중기예보**: 3~10일 예보 (일 2회 갱신)
- **생활기상지수**: 자외선, 체감온도, 대기정체 등

**데이터 범위:**
- 전국 5km × 5km 격자 단위
- 육지 및 해상 데이터
- 과거 기상 데이터 (최대 1개월)

**사용 제한:**
- **호출 제한**: 일 1,000건 (일반), 10,000건 (기업)
- **상업적 이용**: ✅ 가능
- **비용**: 무료

**활용 예시:**
- 날씨 앱 개발
- 농업/축산업 의사결정 지원
- 이벤트/야외활동 계획

---

### 열린국회정보
**제공 데이터:**
- 국회의원 정보 (300명)
- 발의 법률안 및 의안
- 본회의/위원회 회의록
- 국회의원 표결 정보

**데이터 범위:**
- 제1대 국회 ~ 현재 (역대 의정활동 데이터)
- 법안 발의부터 공포까지 전 과정

**사용 제한:**
- **호출 제한**: 일 3,000건
- **상업적 이용**: ✅ 가능 (정치적 목적 제외)
- **비용**: 무료

## 🗺 지도 & 위치 데이터

### 카카오맵 API
**제공 데이터:**
- 대한민국 전역 지도 데이터
- 500만개 이상 장소(POI) 정보
- 주소 검색 및 좌표 변환 (주소 ↔ 위경도)
- 경로 탐색 (도보, 자동차, 자전거)
- 지오코딩/역지오코딩

**데이터 범위:**
- 한국 전역 (서울/수도권 상세, 지방 일반)
- 실시간 교통정보 연계
- 건물명, 도로명, 지번 주소

**사용 제한:**
- **호출 제한**:
  - 지도 표시: 월 300,000회
  - 주소 검색: 일 100,000건
  - 경로 탐색: 일 10,000건
- **상업적 이용**: ✅ 가능 (앱/웹 서비스)
- **재배포**: ❌ 금지 (카카오맵 데이터 자체 판매 불가)
- **비용**: 무료 (일정 한도), 유료 플랜 있음

**사용 불가 사례:**
- 지도 데이터를 다운로드하여 오프라인 저장
- 타사 지도 서비스에 카카오 데이터 오버레이

---

### 네이버 지도 (네이버 클라우드)
**제공 데이터:**
- 전국 상세 지도
- 360도 파노라마 이미지
- Geocoding/Reverse Geocoding
- 경로 탐색 (Directions 5/15)
- Static Map 이미지

**데이터 범위:**
- 한국 전역 (위성사진, 거리뷰 포함)
- 해외 일부 지역 (글로벌 맵)

**사용 제한:**
- **호출 제한**:
  - Geocoding: 일 100,000건 (무료)
  - Directions: 일 5,000건 (무료)
- **상업적 이용**: ✅ 가능
- **비용**: 무료 한도 초과 시 종량제

---

### 브이월드 (국토지리정보원)
**제공 데이터:**
- 3D 지형 데이터
- 정밀 수치지도
- 항공사진/정사영상
- DEM(Digital Elevation Model)
- 건물 3D 모델

**데이터 범위:**
- 대한민국 전역
- 축척별 지도 (1:1,000 ~ 1:25,000)
- 과거 항공사진 (1960년대~현재)

**사용 제한:**
- **호출 제한**: 일 5,000건
- **상업적 이용**: ⚠️ 별도 협의 필요
- **학술/연구**: ✅ 적극 권장
- **비용**: 무료 (상업용 별도)

## 💵 금융 & 결제 데이터

### 금융결제원 오픈뱅킹
**제공 데이터:**
- 19개 은행 통합 계좌 조회
- 잔액 조회
- 거래 내역 (입출금 내역)
- 계좌 이체/송금
- 출금동의 서비스

**데이터 범위:**
- 사용자 본인 계좌만 조회 가능 (OAuth 인증)
- 최대 3개월 거래 내역
- 실시간 잔액 정보

**사용 제한:**
- **호출 제한**:
  - 잔액 조회: 일 500건/사용자
  - 거래 내역: 일 200건/사용자
  - 송금: 일 50건/사용자
- **상업적 이용**: ✅ 가능 (핀테크 서비스)
- **개인정보**: ⚠️ 엄격한 보안 준수 필요 (전자금융법)
- **비용**: 무료 (건당 수수료 발생 가능)

**주의사항:**
- 이용기관 등록 필수 (심사 1~2주)
- 테스트 계좌로 충분한 테스트 후 실서비스
- AML(자금세탁방지) 의무 준수

---

### 한국투자증권 KIS API
**제공 데이터:**
- 국내 주식 시세 (실시간/지연)
- 국내 주식 호가
- 해외 주식 시세 (미국, 중국, 일본 등)
- 주식 주문 (매수/매도)
- 잔고 조회
- 체결 내역

**데이터 범위:**
- 코스피/코스닥 전종목
- 해외 주요 거래소 (NYSE, NASDAQ, 상해, 도쿄 등)
- 실시간 체결가, 호가, 시간외 거래

**사용 제한:**
- **호출 제한**:
  - 실시간 시세: 초당 5건
  - 주문: 초당 2건
- **상업적 이용**: ✅ 가능 (투자 앱 개발)
- **데이터 재판매**: ❌ 금지
- **비용**: 무료 (모의투자), 실투자 시 계좌 개설 필요

**활용 예시:**
- 주식 자동매매 시스템 (퀀트 트레이딩)
- 포트폴리오 관리 앱
- 투자 알림 서비스

## 📊 일반 사용 제한사항

### 공통 제한사항

#### 1. API 호출 제한
대부분의 공공 API는 다음과 같은 호출 제한을 두고 있습니다:

| 구분 | 일반적인 제한 | 초과 시 조치 |
|------|-------------|------------|
| **초당 호출** | 5~10건 | 일시적 차단 (1분) |
| **일일 호출** | 1,000~10,000건 | 24시간 차단 |
| **월간 호출** | 30만건 | 계정 정지 가능 |

**대응 방법:**
- Rate Limiting 구현 (429 에러 처리)
- 캐싱 활용 (Redis, Memcached)
- 배치 처리로 호출 수 줄이기

---

#### 2. 데이터 사용 범위

**✅ 허용되는 사용:**
- 앱/웹 서비스 개발
- 학술 연구 및 분석
- 비영리 프로젝트
- 개인 학습 목적

**⚠️ 주의가 필요한 사용:**
- 상업적 목적 (API별 약관 확인 필수)
- 대량 데이터 수집 (크롤링 금지)
- 제3자 재판매 (대부분 금지)

**❌ 금지되는 사용:**
- 불법적 목적 (스팸, 사기)
- 원본 데이터 그대로 재판매
- 정치적/선거 관련 악용
- 개인정보 무단 수집

#### 4. 개인정보 보호

#### 6. 에러 처리

**일반적인 HTTP 상태 코드:**

| 코드 | 의미 | 대응 방법 |
|-----|------|---------|
| **200** | 성공 | - |
| **400** | 잘못된 요청 | 파라미터 확인 |
| **401** | 인증 실패 | API 키 확인 |
| **403** | 권한 없음 | 접근 권한 확인 |
| **404** | 데이터 없음 | 요청 값 재확인 |
| **429** | 호출 제한 초과 | Rate Limiting 적용 |
| **500** | 서버 오류 | 재시도 (Exponential Backoff) |

## 📌 FAQ: 자주 묻는 질문

### Q1. API 키를 발급받았는데 403 에러가 발생해요.
**A:** 다음을 확인하세요:
- API 키 활성화 여부 (승인 대기 중일 수 있음)
- 요청 도메인/IP 등록 여부
- 요청 헤더에 API 키 올바르게 포함했는지 확인
- Referer 제한 설정 확인

---

### Q2. 상업적으로 사용해도 되나요?
**A:** API별로 다릅니다:
- **무료 허용**: 공공데이터, 네이버/카카오 검색, 지도 등 대부분
- **유료 전환 필요**: CLOVA Studio, ODsay (상업용)
- **별도 계약**: 삼성페이, 삼성 녹스 등 Partnership

API 이용약관을 반드시 확인하세요.

---

### Q3. 데이터를 DB에 저장해도 되나요?
**A:**
- **✅ 허용**: 캐싱 목적으로 일정 기간 저장 (1일~1주)
- **⚠️ 주의**: 대량 저장 후 재판매는 대부분 금지
- **❌ 금지**: 개인정보 장기 저장 (금융, 의료 데이터)

---

### Q4. 호출 제한을 초과하면 어떻게 되나요?
**A:**
- 일시적 차단 (429 Too Many Requests)
- 24시간 후 자동 해제
- 반복 시 계정 정지 가능

**해결책:**
- Rate Limiting 구현
- 유료 플랜 전환
- 캐싱 전략 수립

---

### Q5. 해외에서도 사용할 수 있나요?
**A:**
- **대부분 가능**: 공공데이터, 네이버, 카카오
- **IP 제한 있음**: 일부 금융 API (국내 IP만)
- **지역 제한**: 일부 스트리밍 API

---

### Q6. API 응답이 느려요. 최적화 방법은?
**A:**
1. **캐싱**: Redis/Memcached 도입
2. **비동기 처리**: asyncio, aiohttp 사용
3. **CDN 활용**: 정적 데이터는 CDN에 저장
4. **지역별 서버**: 국내 서버에서 호출
